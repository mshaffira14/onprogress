## Overview
- Dashboard
- Status Akademik
- Pindah Table
- Pindah Presensi
- Dekrip Nilai
- Update KRS


# Features
- Memangkirkan Status Akademik
- Ubah Status Akademik Base On Pembayaran
- Ubah Status Akademik Base On Cuti
- Pindah Table Temp ke Aktiv
- Pindah Aktiv Temp ke Archive
- Pindah Nilai
- absensi_mhs_aktiv -> absensi_mhs_archive
- absensi_meet_aktiv -> absensi_meet_archive
- absensi_mhs_ijin -> absensi_mhs_ijin_archive
- krs_jadwal_hadir -> krs_jadwal_hadir_archive
- krs_prosentasi_nilai_aktiv -> krs_prosentasi_nilai_archive
- krs_persen_hadir -> krs_persen_hadir_archive
- krs_validasi_nilai_aktiv -> krs_validasi_nilai_archive
